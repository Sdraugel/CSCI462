Welcome to the TSD repository

To run the project you need to:

  • download the zip
  • unpack and locate install.bat
  • run install.bat (it will install all of the required software)
  • locate the TSD project in your C directory
  • run the exe

Required software:

  • Python 2.7 or higher
  • Windows 7
